### 一、深圳科目三标准流程（两步上车法）

**第一步：初次上车（验证身份）**

1. 走到考试车旁，**开门上车**。
2. 调整座椅、后视镜（先调好，避免绕车回来再调）。
3. 递交身份证，**指纹 / 人脸识别**验证身份。
4. **等待系统语音**：**"请开始考试" 或 "上车准备"**。

**第二步：听到指令后 → 下车绕车一周**

- **必须听到语音后再下车**，提前下车绕车无效、直接挂科。
- **逆时针**绕车一周（从主驾门 → 车尾 → 右后 → 右前 → 车头 → 左前 → 回到主驾）。
- 经过**感应点**时（一般 4 个：左后、右后、右前、左前），**用力按 2 秒**，等语音提示 "检测到" 再走。

**第三步：绕车完成 → 第二次上车**

1. 观察后方交通，确认安全开门上车。
2. **关紧车门**（没关严直接不合格）。
3. **系安全带**、检查：**空挡、手刹、灯光复位**。
4. 等待**模拟夜间灯光考试** → 起步。
