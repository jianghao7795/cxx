## 一、文件树 ↔ 编辑窗口（跨窗口焦点切换）

### 1. 原生通用（所有窗口通用，必记）

```
<C-w> h   光标切到左侧（文件树）
<C-w> l   光标切到右侧（编辑文件）
<C-w> w   循环切换所有窗口（一键来回）
```

简写优化（LazyVim 默认映射，更顺手）：

```
<C-h>  = <C-w>h  左跳文件树
<C-l>  = <C-w>l  右跳编辑区
```

> 操作流程示例：
>
> 1. `<space>e` 打开左侧文件树
> 2. 选中文件按 `l` 打开（光标跳到右侧编辑）
> 3. 想切回文件树：直接按 `<C-h>`
> 4. 切回代码：`<C-l>`

### 2. neo-tree 专属一键跳转（树面板内快速回代码）

在文件树面板里，直接按 `<C-l>` 立刻回到编辑窗口，不用敲 `<C-w>`。

## 二、文件树内部：文件夹与文件导航（neo-tree 面板内）

光标停在左侧树形面板时：

表格

| 按键 | 功能                                      |
| :--- | :---------------------------------------- |
| `l`  | 展开文件夹 / 打开选中文件（进入文件编辑） |
| `h`  | 折叠文件夹 / 返回上一级父目录             |
| `j`  | 向下移动光标（切换下一个文件 / 文件夹）   |
| `k`  | 向上移动光标（切换上一个文件 / 文件夹）   |
| `r`  | 重命名                                    |
| `a`  | 新建文件 / 文件夹                         |
| `d`  | 删除文件 / 文件夹                         |

示例：

- 光标在文件夹上 → `l` 展开子文件；
- 光标在文件上 → `l` 打开文件，光标自动切到右侧编辑区；
- 想退回上层目录：光标在文件夹最外层，按 `h`。

## 三、打开 / 关闭文件树（快速呼出文件夹面板）

默认 `<leader` 是空格 `space`

```
<space> e   打开项目根目录文件树（切换显示/隐藏）
<space> E   打开当前文件所在目录文件树
```

- 编辑代码时按 `<space>e`：左侧弹出文件夹树；
- 再按一次 `<space>e`：关闭文件树，全屏编辑。

## 四、多个已打开文件之间切换（buffer 切换，不用文件树）

打开多个文件后，在编辑区快速切换，不用切左侧树：

1. 左右切换缓冲区

plaintext









```
Shift + h  上一个打开的文件
Shift + l  下一个打开的文件
```

1. 列出所有打开文件选择

plaintext









```
<space> ,    打开buffer列表，上下选择文件回车切换
<space> bb   在两个最近文件之间来回切换
```

## 五、常用完整工作流

1. 打开项目文件树：`<space>e`
2. 在树里上下选目录 / 文件：`j/k`
3. 展开文件夹：`l`；返回上级：`h`
4. 打开文件自动跳到编辑区：`l`
5. 编辑中途切回文件树：`<C-h>`
6. 切回代码编辑：`<C-l>`
7. 关闭文件树：`<space>e`
8. 多文件来回切：`Shift h/l`
